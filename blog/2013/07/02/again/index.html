Again!

Jul 2nd, 2013

Trying this Octopress thing again…

I guess I’m continually drawn to static site generators. I recently listened to The Web Ahead podcast, and the topic was Jekyll. And, of course, I was intrigued at the thought of a system that can manage content similar to a familiar CMS (like WordPress in my case), with loops and includes and templates, but whose end product is static HTML, CSS and JavaScript files.

The thought of not relying on a database and a server running thousands of lines of PHP code to pull everything together as it is requested is very appealing.

The part I’m not sure about is whether to look into Jekyll, or to keep on with Octopress. Octopress seems like a head start, but so far I have found myself not liking it in the same way that I don;t like starting a WordPress site with an overly developed theme.

I really hate spending any amount of time deconstructing and re-engineering a template to suit my needs. For now, I have Octopress set up here, so I’ll continue checking it out and see if I can’t find a more “blank” starting point…
